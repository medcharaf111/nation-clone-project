# Arabic Landing Page

This project is a landing page designed to present an application in Arabic. It serves as an introduction to the app before users are directed to the terms and conditions.

## Project Structure

```
arabic-landing-page
├── public
│   └── index.html        # Main HTML document for the landing page
├── src
│   ├── css
│   │   └── styles.css    # Styles for the landing page
│   ├── js
│   │   └── main.js       # JavaScript for dynamic content and interactions
│   └── locales
│       └── ar.json       # Localization strings in Arabic
├── package.json           # npm configuration file
└── README.md              # Project documentation
```

## Setup Instructions

1. **Clone the repository:**
   ```
   git clone <repository-url>
   cd arabic-landing-page
   ```

2. **Install dependencies:**
   ```
   npm install
   ```

3. **Run the application:**
   ```
   npm start
   ```

## Usage

- Open the `public/index.html` file in a web browser to view the landing page.
- The page is designed to be fully responsive and provides information about the app in Arabic.

## Localization

The project includes an `ar.json` file located in the `src/locales` directory, which contains all the necessary localization strings for the Arabic language. This allows for easy updates and translations.
